# Blog de Café

## Descripción del Proyecto
Este proyecto es un **Blog de Café** desarrollado con **HTML**, **Bootstrap 5**, **CSS**, y **JavaScript**. Diseñado para los amantes del café, ofrece artículos, recetas y noticias sobre el mundo del café, todo presentado de manera atractiva y fácil de navegar.

## Características
- **Diseño Responsivo**: Optimizado para todos los dispositivos, desde móviles hasta escritorios.
- **Interactividad**: Se utilizan componentes de Bootstrap y JavaScript para una mejor experiencia de usuario.
- **Sección de Artículos**: Publica entradas sobre diferentes temas relacionados con el café, como métodos de preparación y variedades de granos.

## Estructura del Proyecto
La aplicación está organizada en varias secciones:
- **Página Principal**: Presenta las últimas entradas del blog y destaca artículos populares.
- **Página de Nosotros: Ofrece información sobre la temática del blog y el equipo detrás de él
- **Página de Contacto: Permite a los usuarios enviar consultas y sugerencias, además de ofrecer una colección de recetas fáciles de seguir.
